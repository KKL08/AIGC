# Comparison — Creativity Map | 领域指南

基于 60 条 EvoLinkAI 高质量 prompt 提炼而来的 Comparison 领域指南。

## 方向 1: 游戏截图仿真 (约 8 条)

**视觉特征:** 模拟真实游戏内截图的画面风格，包含HUD/UI覆盖层、特定游戏的视觉语言(像素/3D/写实)、游戏跨界mashup
**关键技巧:**
- 极简指令: "an ingame screenshot of rust" / "AmongUsの精密な実際のゲーム画像を生成して"
- 跨界混搭: counter strike mixed with Terraria，将两种游戏风格融合
- UI 完整度: 完整的战斗界面(HP/SP/BURST meter)、技能菜单、物品栏、任务面板、小地图
- 幽默化: 将现实场景(东京迪士尼抢前排)包装为格斗游戏UI，制造荒诞感

**代表性 prompt (case6):**
> counter strike in game screenshot, mixed with Terraria

**覆盖条目:** case1, case5, case6, case7, case10, case12, case23, case47

## 方向 2: 中国历史文化艺术 (约 12 条)

**视觉特征:** 中式美学(工笔重彩/新中式极简/东方幻想)、历史人物写实肖像、古装武侠电影感、低饱和高级配色
**关键技巧:**
- 古典重现: 工笔重彩长卷(唐代长安城元宵灯会全景)、写实佛教/历史人物肖像
- 新中式极简: 巨大荷花作为空间容器+微缩城市，低饱和干净粉色，暖米白宣纸质感，大面积留白
- 武侠电影: 9:16竖版、冷月光+蓝光剑气、湿身水光效果、衣袂猎猎、dramatic atmosphere
- 东方幻想: 半透明丝绸礼服、鎏金纹理、暖金色逆光、光粒漂浮、artstation热门作品风格
- 暗黑神话: 中式怪异+黑暗神秘+多重管线渲染，近地仰拍镜头+强烈阴影

**代表性 prompt (case25):**
> 新中式极简东方美学 × 高端商业插画，主题一花一世界，极简，克制，空灵，高级商业视觉，超现实东方意境... 一朵巨大的荷花作为空间容器，从平静水面自然生长... 荷花内部为唯一视觉焦点：发光的3D微缩广州城市...

**覆盖条目:** case5, case24, case25, case26, case27, case29, case30, case31, case32, case33, case35, case37, case38

## 方向 3: 分镜脚本与动画流程 (约 8 条)

**视觉特征:** 16:9白底分镜海报、8-9 panel网格、Pixar 3D/手绘/电影风、每panel含时间码+镜头说明+动作描述、角色一致性
**关键技巧:**
- 分镜结构: 8-9 shots, timecode per panel, shot type + camera move + action note
- 角色一致性: same Pixar-style character throughout all panels，recurring secondary character
- 叙事弧线: opener → build → hero frame → reveal → closing，完整短片节奏
- 制作标注: footer含VIDEO FLOW/CAMERA TIPS/LIGHT & STYLE等制作指引
- 风格变体: Pixar 3D(case56,57,59) / 手绘sketch(case53) / 电影写实(case58,60)

**代表性 prompt (case56):**
> Title: "THE DONUT HEIST" Subtitle: "15s Comedic Storyboard – 9 Shots" Style: full-color 3D animated storyboard sheet, cinematic cartoon characters... Each panel: timecode, shot type, camera move, short action note. 01 0–1.5s Wide/static: raccoon finds donut box on park bench...

**覆盖条目:** case48, case53, case56, case57, case58, case59, case60

## 方向 5: 创意技法转化 (约 9 条)

**视觉特征:** 将照片/概念转化为特定视觉形式——博物学标本解剖、涂色填数、拍立得突破、POV视角切换、cozy scrapbook
**关键技巧:**
- 博物学标本: 食物剖面解剖，卡拉瓦乔光影 × 博物插画，黑丝绒背景，手写标注(结构名/成分/意义)
- 涂色填数(Paint by Numbers): 自适应颜色数(1-14)，闭合区域+数字标注，底部色板图例
- 视角切换: 同一场景两图对比(Source/Output)，仅改变camera POV
- 拼贴变形: scrapbook-style(chibi mini alter egos)、folk doodle simplification
- 设计工具: 室内设计Mood Board生成器、视觉品牌拆解图

**代表性 prompt (case40):**
> 一颗/一块/一枚【食物名称】，以博物学大师发现野外标本的方式解剖。剖开、展开、固定——如同博物馆的珍贵藏品，却以卡拉瓦乔为《国家地理》掌镜时的光线照亮... 整体气质：奥杜邦博物插画×卡拉瓦乔光影×有史以来最美的科学摄影。

**覆盖条目:** case40, case41, case42, case43, case49, case51, case52, case54, case55

## 方向 6: 叙事概念海报 (约 5 条)

**视觉特征:** 轮廓宇宙/收藏版叙事海报、游戏对战海报、科幻概念、IP集结图，高审美+强叙事
**关键技巧:**
- 轮廓宇宙: 主轮廓(建筑/雕像/面具等)内部承载完整叙事世界，水彩质感+纸张印刷品气质
- 审美标准: 第一眼主题识别度→第二眼叙事世界→第三眼细节余味
- 色彩克制: 低饱和、高级、耐看，黑金灰/冷蓝灰/褐红米白等体系
- 负面约束: 不要游戏宣传图感、不要过度卡通化、不要模板化奇幻素材

**代表性 prompt (case4):**
> 请根据【主题：xxx】自动生成一张高审美的"轮廓宇宙 / 收藏版叙事海报"风格作品。不要将画面局限于固定器物或常见容器... 优先选择最能放大主题气质、最能形成强烈视觉记忆点、最能体现史诗感、神秘感、诗意感或设计感的轮廓...

**覆盖条目:** case4, case9, case34, case39, case50

## 方向 7: 能力测试与趣味生成 (约 8 条)

**视觉特征:** 简单直接的指令测试AI图像能力——计数/细节/文字渲染/历史事件/名人场景/仿照编辑
**关键技巧:**
- 计数测试: 书架上精确数量的书籍(1/3/7)
- 细节展示: 四季眼部特写超写实，每屏不同美瞳+睫毛装饰+季节元素
- 名人场景: Sam Altman riding a bear selfie, 名人直播截图
- 事件生成: 生成2020/2001年最重大事件图像
- 编辑能力: 修改图中金额数字、去背景透明化

**代表性 prompt (case2):**
> 以眼部特写图片为基础，生成3:4的四屏构图超写实眼部特写，四屏按春夏秋冬上下排序。第一屏：眼眸中带着绽粉樱色的美瞳，睫毛缀满迷你春花...

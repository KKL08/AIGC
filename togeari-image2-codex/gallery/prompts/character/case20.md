**Author:** @icreatelife

## Prompt

Transform the provided photo into 6 different characters in the style of 2000s cartoon TV shows. Each character should capture a distinct cartoon show aesthetic from that era while maintaining recognizable features from the original photo. Arrange in a 2x3 or 3x2 grid format with the cartoon show name labeled under each. Clean white background between panels.

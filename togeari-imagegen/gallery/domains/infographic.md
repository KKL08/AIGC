# Infographic — Creativity Map | 领域指南

基于 22 条高质量 prompt 提炼而来的 Infographic 领域指南。

---

## 方向 1: 教育科普型结构图 (约 6 条)

**视觉特征:** 清晰的分区布局（上中下或左右列），每个区块承载独立知识点；大量使用编号标注、箭头流向、图例和小结框；整体以浅色干净背景为主，色彩编码用于区分不同概念层级而非纯粹装饰。核心目标是让复杂知识"一张图看懂"。

**关键技巧:**
- `section-numbering`: 用"一、二、三"或"01 02 03"等编号串联知识逻辑，给读者清晰的阅读路径
- `color-coded zones`: 对不同数据类别/阶段赋予固定颜色，让视觉分层取代文字说明
- `legend + callout labels`: 箭头方向图例搭配引线标注，让图示自说明，减少冗余文字
- `footer summary`: 底部用"小结"/"核心结论"模块收尾，强化记忆锚点
- `multi-column grid`: 顶部说明 + 核心图示 + 底部实例三段式布局，适合知识密度高的主题

**代表性 prompt (case1):**
> Create an educational infographic in a clean, modern textbook illustration style with a light blue and white color palette. Header: "高中地理：大气环流的示例说明". Layout: top-left column with Section 1 "一、什么是大气环流？"... Section 3 with Earth cross-section showing latitudes, color-coded pressure belts, arrows indicating air flow.

**覆盖条目:** case1, case4, case10, case13, case14, case21

## 方向 2: 文化器物博物馆图录风 (约 4 条)

**视觉特征:** 模拟国家级博物馆展板的专业排版——米白/丝白/浅茶色纸张纹理背景，真实主体物大图居中，四周散布结构拆解、工艺流程、材质纹理样本、纹样寓意图解。所有文字为清晰可读的中文，刻意规避商业海报感与影棚感，追求历史文献的庄重感与知识密度。

**关键技巧:**
- `object-centered layout`: 主题物件占据版面核心，其余模块围绕其展开，形成"物件即锚点"的阅读体验
- `material texture samples`: 在版面嵌入真实纹理局部特写，让材质说明从文字变成视觉证据
- `annotation leader lines`: 引线从主体延伸至标注框，标注框层级分明（部件名 + 说明 + 工艺）
- `timeline integration`: 横向时间轴嵌入历史演变脉络，为静态器物注入时间维度
- `parchment/paper background`: 纸质纹理背景赋予"文献感"，区别于数字信息图的玻璃/渐变美学

**代表性 prompt (case11):**
> 整张图需要把真实主体主视觉、结构拆解、中文标注、材质说明、纹样寓意、色彩寓意和核心特征总结融合在一起。整体风格应偏向国家级博物馆展板、历史服饰图录和文化博物馆专题信息图。背景使用米白、丝白、浅茶色等纸张纹理。

**覆盖条目:** case11, case12, case9, case2

## 方向 3: 产品技术拆解图 (约 4 条)

**视觉特征:** 以产品/设备为核心，通过爆炸图（exploded view）、等轴测图或分层剖面图展示内部结构；高科技渲染质感（3D render、工作室打光、渐变背景、发光细节）搭配两侧整齐排列的标注文字。视觉语言借鉴工业设计文档与高端品牌产品手册。

**关键技巧:**
- `exploded-view layering`: 各组件沿垂直或斜轴方向分离展开，保持空间关系的同时让每层独立可见
- `bilateral callout labels`: 左右两侧对称分布标注，引线指向各组件，保持版面平衡
- `studio lighting + glow accents`: 工作室光源搭配局部发光细节，强化高科技感与精密仪器的质感
- `brand identity integration`: 底部页脚嵌入 headline + 正文块 + 品牌 logo，使技术图具备商业发布级完成度
- `bento grid modules`: 用 8 格或更多模块分别展示产品不同维度（主图、规格、颜色、尺寸比较），适合电商/发布会场景

**代表性 prompt (case7):**
> Create an exploded view product diagram poster for a VR headset. Clean high-tech 3D render with studio lighting and glowing accents. Soft purple and blue gradient background. Centerpiece: vertically stacked exploded view showing 9 distinct layers of internal components... 8 callout labels on left and right sides.

**覆盖条目:** case5, case7, case8, case20, case22

## 方向 4: 手绘地图与旅行图鉴 (约 3 条)

**视觉特征:** 以地理空间为骨架，用水彩+墨线手绘风格将地图转化为插画叙事场景；地标建筑以等轴测或立面手绘形式呈现，道路、河流、公园各有色彩编码；食物/景点以卡通插图分布于地图之上，饱含生活气息与在地文化性格。复古羊皮纸底色或明亮杂志配色二选一，前者强调历史感，后者强调传播力。

**关键技巧:**
- `watercolor + ink linework`: 水彩晕染打底，墨线勾勒轮廓，手工质感胜过数字平滑
- `illustrated landmark icons`: 建筑/场所以手绘小插图呈现，不用真实照片，保持整体风格统一
- `mascot + decorative border`: 主题吉祥物（熊猫、辣椒）配合藤蔓/花边框，强化在地个性
- `vintage compass rose + parchment`: 复古罗盘搭配羊皮纸纹理，营造"探索者手记"的叙事氛围
- `modular day-color zoning`: 旅行指南版将三天行程用色块分区，每天一色，视觉上即导航

**代表性 prompt (case6):**
> Create an illustrated map infographic in watercolor and ink hand-drawn style on vintage parchment. Title: "成都 吃货暴走地图" with cartoon red chili pepper mascot wearing sunglasses. Border of vine leaves and red chili peppers. Background: textured beige parchment with yellow roads, blue rivers, green park areas. 6 landmarks illustrated...

**覆盖条目:** case6, case15, case21

## 方向 5: 文件仿真与时代穿越 (约 3 条)

**视觉特征:** 核心技法是"以假乱真的文件感"——模拟真实世界的历史文件（专利档案、医院处方、高考试卷），或将现代界面植入古代语境（宋朝朋友圈）。成功的关键在于细节的真实性：纸张老化痕迹、手写笔迹、官方印章、规范表格格式，或社交媒体 UI 元素与古典纹样的精准混搭。

**关键技巧:**
- `document texture simulation`: 老化纸张（foxing、折痕、泛黄）或医院/考场纸张的印刷肌理，是"真实感"的第一道关卡
- `authentic typography mirroring`: 字体、字号、行距严格对标真实文件类型，手写字体要"自然随意但可辨认"
- `official seals + signatures`: 印章、签名、编号等权威符号是仿真文件的可信度锚点
- `UI skin transplant`: 把古代元素（宋代纹样、文人画像）逐一替换现代 UI 组件（点赞图标、状态栏），而非简单叠加，是时代穿越类成立的关键
- `overhead photography framing`: 从上方俯拍"放在桌上的文件"，摄影视角本身就是真实感的来源

**代表性 prompt (case9):**
> A vintage patent document for an invention, styled after late 1800s United States Patent Office filings. The page features precise technical drawings with numbered callouts (Fig. 1, Fig. 2, Fig. 3)... The paper is aged ivory with foxing stains and soft fold creases. An official embossed seal and red wax stamp appear in the corner.

**覆盖条目:** case9, case17, case18, case19

## 方向 6: 品牌系统与叙事关系图 (约 2 条)

**视觉特征:** 这类信息图的内容本身就是"设计系统"——品牌规范手册（logo构造、色板、字体、应用场景）或角色关系图谱（用线型/颜色编码多维关系网络）。版面高度结构化，网格系统严格，视觉元素既是内容也是示例。整体呈现专业品牌手册或高设计感出版物的完成度。

**关键技巧:**
- `golden ratio grid construction`: logo 构造网格用黄金比例标注，展示设计决策的逻辑性而非结果
- `color specification with hex swatches`: 色板附带 HEX 值和最小使用规范，而非仅展示颜色本身
- `incorrect usage examples`: 5 种错误用法示例让规范从"应该这样"变为"绝不能那样"，信息密度倍增
- `relationship line taxonomy`: 用不同颜色+线型（实线/虚线/波浪线）编码不同关系类型，搭配图例，让复杂网络可读
- `style DNA extraction`: 从原作中提炼配色、材质、符号等视觉基因，注入信息图的整体氛围，使图本身成为对原作的延伸

**代表性 prompt (case3):**
> Create a complete brand identity system board for "狗东西 (GDX)" pet brand. Include: logo construction grid (golden ratio 1.618:1), 4 concept sketch evolution stages... color specification with 5 hex swatches (#1E3D34, #F5F3EF...), typography specs with font samples, minimum size rules, clear space diagram, and 5 incorrect usage examples.

**覆盖条目:** case3, case16
